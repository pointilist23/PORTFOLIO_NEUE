Aerocchat.

Graphic Design.

Aerochat is a third party Discord Client made by nullptr which replicates the look of 2009's MSN Messenger 1:1.
The project started originally as WebApp, however it underwent a rewrite to run natively under WPF, which allowed for faster speeds and far easier implementing of custom assets and user customisation options.
This is where my role came in. I was tasked making new custom, but period-realistic "scenes" that people could use as their user banner. These scenes had to fit a strict size format and be simultaneously generic, but appealing to users of the software who would want to choose these to represent themselves.
When the relaunch happened, these scenes were originally the only unique customisation additions to the software, however we later decided to allow the community to submit their own designs, and for this I also made a Scene Previewer and user guidelines template.
